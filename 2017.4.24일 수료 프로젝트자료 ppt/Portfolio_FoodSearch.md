## 💡 기획 배경

### 외식업 시장 트렌드 (2016년)

- **혼밥/혼술 보편화**: 1인 가구 증가, 혼자 외식 빈도 2015년 대비 **32% 이상 상승**
- **가성비 추구**: 경기불황 지속 → 1번의 식사도 맛있게 먹고 싶은 니즈
- **푸드 서비스 테크의 진화**: 요식업 + IT 기술 접목 (배달앱, 빅데이터, 전자결제 등)

### 기존 서비스의 문제점

| 문제 | 설명 |
|------|------|
| 🔍 검색 데이터 부족 | 찾아도 끝없이 같은 결과 (동일 블로그) |
| 📊 객관적 평가 부재 | 조회순/추천순 등 신뢰할 수 있는 평가 시스템 없음 |
| 💰 과도한 광고 | 이윤 극대화를 위한 무절제한 광고 배너 |
| ❌ 부정확한 정보 | 객관적이지 못한 바이럴 마케팅 자료 |

### 솔루션

> **사용자 기반의 객관적 맛집 평가 시스템**을 통해 신뢰할 수 있는 맛집 검색 서비스를 제공

## 🗄 DB 설계

프로젝트에서 사용된 주요 데이터베이스 테이블:

- 회원 관리 (사용자 정보, 인증)
- 식당 정보 (상세 정보, 키워드, 메뉴)
- 리뷰 시스템 (리뷰, 추천, 댓글)
- 좋아요 리스트
- 게시판 (공지사항, 일반 게시판)
- 검색 관련 (인기 검색어 등)

## ⭐ 주요 기능

### 1. 회원 관리
| 기능 | 설명 |
|------|------|
| 로그인 | 팝업 로그인, 패스워드 체크 |
| 회원가입 | 유효 이메일 체크, 전화번호/생년월일/성별/대표이미지 입력 (유효성 검사) |
| PW 찾기 | 이메일 + 생일 기반 비밀번호 찾기 |
| 마이페이지 | 이메일, 생년월일, 전화번호, 성별, 닉네임, 관심음식, 자기소개, 회원등급, 포인트 관리 |

### 2. 메인 페이지
| 기능 | 설명 |
|------|------|
| 동영상 배경 | 상단 동영상 배경 + 스크롤 시 고정 배너 |
| 검색창 | 스크롤 시 상단 배너에 검색창 자동 표시 |
| 인기 검색어 | 검색창 클릭 시 Ajax로 인기 검색어 로딩 |
| 자동완성 | 검색어 입력 시 자동완성 기능 |
| 식당 등록 | 모달 창으로 사진 포함 식당 등록 |

### 3. 상세 검색
| 기능 | 설명 |
|------|------|
| 지도 연동 | **Naver Map API** — 검색 결과 마커 표시, 자동 이동, 다중 마커 |
| 필터 검색 | 장소, 음식 종류, 가격대, 예약/주차장 여부 |
| 지역 선택 | 지역별 선택 / 다중 선택 가능 |
| 페이징 | 검색 리스트 페이징 처리 |
| 좋아요 리스트 | 로그인 시 좋아요 누른 맛집 지도 마커로 표시 |

### 4. 식당 상세 정보
| 기능 | 설명 |
|------|------|
| 상세 정보 | 식당 이미지, 한줄 설명, 대표 메뉴 사진, 키워드, 위치 지도 |
| 리뷰 시스템 | 모달 리뷰 작성, 수정/삭제, 추천 기능 |
| 정렬 | 최신순/추천순 리뷰 정렬 |
| 좋아요 | 좋아요 리스트 추가/삭제 |

### 5. 게시판
| 기능 | 설명 |
|------|------|
| 기본 CRUD | 게시글 작성/수정/삭제 |
| HOT 표시 | 조회수 20 이상 시 'HOT' 배지 |
| 페이징 | 10건씩 페이징 처리 |
| 검색 | 게시판 내 검색 기능 |
| 관리자 기능 | 체크박스 일괄삭제 / 선택삭제 |

## 🏅 성과 및 회고

### 기술적 성과

| 영역 | 내용 |
|------|------|
| **Spring MVC** | Dispatcher Servlet 기반 MVC 아키텍처 구현 |
| **Ajax 비동기 처리** | 인기 검색어 로딩, 자동완성, 좋아요 기능 |
| **Naver Map API** | 지도 연동, 다중 마커, 자동 이동 |
| **DB 설계** | Oracle 기반 관계형 데이터베이스 설계 |
| **반응형 UI** | Bootstrap + jQuery 기반 동적 UI |
| **게시판 시스템** | 페이징, 검색, 권한별 관리 기능 |

### 핵심 학습 포인트

1. **Spring MVC 패턴**의 실제 프로젝트 적용 경험
2. **외부 API (Naver Map)** 연동을 통한 지도 기반 서비스 개발
3. **Ajax** 기반 비동기 데이터 처리와 사용자 경험 향상
4. **Oracle DB** 설계 및 CRUD 작업 최적화
5. **팀 협업** 및 역할 분담을 통한 프로젝트 관리 경험
